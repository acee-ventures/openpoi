# SSF 产品说明

> SF South Bay — 可售产品类型与购买流程

---

## 1. 产品概览

SSF 提供两条产品线：

- **买家线 (Stripe)**: 使用权、选择权等功能性产品
- **投资线 (USDC)**: SSFShareToken — 项目参与及功能性凭证

| 参数           | 值                           |
| -------------- | ---------------------------- |
| **项目**       | SF South Bay (70 套住宅单元) |
| **网络**       | Base Mainnet (ChainID: 8453) |
| **Series Cap** | 20,000,000 USDC              |

---

## 2. 买家线（Stripe）

### A1. Any-Unit Select Pass（任选权）

- **卖点**: 先买"选择权"，后在选房窗口期从库存中任选一套
- **交付**: 选房窗口（批次/时间）+ 优先级 + 锁定规则
- **支付**: Stripe

### A2. Annual Stay Pass（年度可用天数）

- **卖点**: 绑定指定 unitId，每年可住/可用 X 天；可自用或委托出租未用天数
- **天数档位**: 30 / 60 / 90 天
- **预订规则**: 旺季/黑名单日期/最短入住
- **未用天数处理**: 不滚动或仅滚动 1 个周期
- **支付**: Stripe

### A3. Swap / Exchange Right（置换权）

- **卖点**: 允许将已锁定/已绑定的 unit 置换到另一 unit
- **规则**: 差价 + 固定手续费，每年度最多 1 次
- **支付**: Stripe

### A4. Rent-Out Opt-In（委托出租授权）

- **卖点**: 把未用天数委托给出租池运营，按规则获得净租金分成
- **分成口径**: 见净租金口径
- **支付**: 默认不单独收费

---

## 3. 投资线（USDC）

### B1. SSFShareToken（项目参与及功能性凭证）

- **定位**: 高端参与及功能性凭证 (premium participation & utility vehicle)
- **总供应**: 20,000 SSFShareToken (Series Cap: 20,000,000 USDC)
- **发行价**: 1,000 USDC per Share
- **Floor Price**: 500 USDC (Reserve Redemption Feature，非保证性退出机制)
- **支付**: USDC（Base Mainnet）
- **发行方式**: Reg D 506(b) 私募，Invite-only
- **资格**: 100% Accredited Investor, Allowlist 上限 100 地址
- **功能性权益**:
  - Stake-to-Access: 解锁优先选房窗口与专属数据室
  - Priority Gating: 未来资产管线优先权
  - Discounts & Rebates: 质押解锁可审计折扣
- **资金隔离 (50/450/500)**:
  - P1 Reserve Vault: 50 USDC (5%) — 退出机制
  - P2 Owner Proceeds: 450 USDC (45%) — T+0 结算
  - P3 SPV Joint Vault: 500 USDC (50%) — 运营 + 激励

> **合规锚点**: 公司不分配股息、租金收入或现金收益。激励为自由裁量的 $POI 奖励，不保证流动性或价值。

## 4. 购买流程

### Web2（Stripe）

1. 用户访问项目页面 `/p/sf-south-bay`
2. 选择产品类型与档位
3. 确认条款版本、风险披露与隐私政策
4. 通过 Stripe 完成支付
5. 系统生成订单回执（含条款版本、披露确认、审计信息）
6. 用户获得相应权益

### Web3（USDC / Base Mainnet）

1. 用户完成 KYC/白名单审核
2. 连接钱包（支持 Base Mainnet）
3. 授权 USDC 额度
4. 调用合约 `deposit(termDays, amount)`
5. 交易确认后获得份额凭证
6. 季度分配时调用 `claim()` 领取

## 5. 净租金口径

```
Gross Rent（毛租金）
− 物业管理费
− 清洁/周转费用
− 维修/保险/税费准备金
= Net Rent（净租金，分配基数）
```

## 7. 分配与置换

### 季度分配

- 每季度末进行分配计算
- Provider 出具确认函/系统通知
- 季度结算后计入可领取余额
- 用户调用 `claim()` 函数领取 USDC

### 置换规则

- 每年最多 1 次
- 费用 = 差价 + 固定手续费
- 需提前申请并经 Provider 确认
- 受限于库存可用性

---

## 8. 对账口径

| 账本            | 用途                               | 分离原则        |
| --------------- | ---------------------------------- | --------------- |
| **Stripe 账本** | 选择权/使用权/置换/订阅/管理服务费 | Web2 权利与服务 |
| **USDC 账本**   | 投资认购与结算（分配、赎回）       | 链上可验证      |
| **POI**         | 仅 stake 产生权限                  | 不作为支付手段  |

- 两套账本独立维护与导出
- 支持 CSV 导出 + SHA256 校验
- 链上交易可在 Base Mainnet 区块浏览器验证
