# SF South Bay 术语表

## 开发阶段术语

| 术语         | 英文                          | 定义                                           |
| ------------ | ----------------------------- | ---------------------------------------------- |
| 地块控制     | Site Control                  | 通过购买协议、期权或其他方式获得对土地的控制权 |
| 规划审批     | Entitlements                  | 获得政府批准的土地使用权和开发许可             |
| 环境影响报告 | EIR                           | 评估项目对环境影响的报告，通常需要公众评议     |
| 建筑许可     | Building Permit               | 政府颁发的允许开始施工的许可证                 |
| 入住许可     | Certificate of Occupancy (CO) | 确认建筑符合所有法规、可以入住的证书           |

## 文件类型

| 术语       | 英文          | 定义                           |
| ---------- | ------------- | ------------------------------ |
| 购买意向书 | LOI           | 表达购买意向的非约束性文件     |
| 购买协议   | PSA           | 具有法律约束力的购买合同       |
| 尽职调查   | Due Diligence | 对项目进行全面调查和评估的过程 |
| 产权调查   | Title Search  | 核实土地产权清晰、无纠纷的调查 |

## 规划术语

| 术语       | 英文              | 定义                                        |
| ---------- | ----------------- | ------------------------------------------- |
| 容积率     | FAR               | 建筑总面积与土地面积的比率                  |
| 建筑覆盖率 | Building Coverage | 建筑底面积占土地面积的比例                  |
| 分区       | Zoning            | 政府对土地用途的分类和限制                  |
| 混合用途   | Mixed-Use         | 同一建筑或区域内包含多种用途（如住宅+商业） |

## 施工术语

| 术语     | 英文                             | 定义                       |
| -------- | -------------------------------- | -------------------------- |
| 总承包商 | GC                               | 负责整体施工管理的承包商   |
| 分包商   | Subcontractor                    | 负责特定工种的承包商       |
| 变更单   | Change Order                     | 对原始合同范围的修改       |
| MEP      | Mechanical, Electrical, Plumbing | 机械、电气、管道系统的统称 |

## 风险管理术语

| 术语     | 英文             | 定义                     |
| -------- | ---------------- | ------------------------ |
| 严重度   | Severity         | 风险发生后的影响程度     |
| 可能性   | Likelihood       | 风险发生的概率           |
| 缓解措施 | Mitigation       | 降低风险影响或概率的行动 |
| 应急计划 | Contingency Plan | 风险发生时的应对方案     |

## 项目阶段

| 术语    | 定义                    |
| ------- | ----------------------- |
| Phase 1 | 地块控制阶段 (已完成)   |
| Phase 2 | 规划审批阶段 (当前阶段) |
| Phase 3 | 建筑许可阶段            |
| Phase 4 | 施工阶段                |
| Phase 5 | 交付阶段                |
