## 风险详情

### RISK-001: 公众听证会反对意见

| 属性       | 值                                           |
| ---------- | -------------------------------------------- |
| **类别**   | 审批                                         |
| **严重度** | 🔴 高                                        |
| **可能性** | 中                                           |
| **影响**   | 可能导致项目延期 2-3 个月                    |
| **缓解**   | 提前社区沟通、详细影响缓解方案、预留时间缓冲 |

### RISK-002: 施工成本上涨

| 属性       | 值                                                  |
| ---------- | --------------------------------------------------- |
| **类别**   | 财务                                                |
| **严重度** | 🔴 高                                               |
| **可能性** | 高                                                  |
| **影响**   | 材料和人工成本上涨，预计 5-10% 超支风险             |
| **缓解**   | 提前锁定主要材料价格、预留 10% 应急预算、分阶段招标 |

### RISK-003: EIR 补充要求

| 属性       | 值                                                 |
| ---------- | -------------------------------------------------- |
| **类别**   | 审批                                               |
| **严重度** | 🟡 中                                              |
| **可能性** | 中                                                 |

| **缓解**   | 与环境顾问密切沟通、主动识别潜在问题、准备备选方案 |

### RISK-004: 分包商资源紧张

| 属性       | 值                                       |

### RISK-006: 地下管线发现
