# SSF Token Catalog

> SSF 项目代币与凭证分类说明

---

## 1. 核心代币 (Core Tokens)

### $POI — Proof of Influence

| 属性       | 值                                                |
| ---------- | ------------------------------------------------- |
| **类型**   | ERC-20 (ERC20Votes, UUPS)                         |
| **网络**   | Base Mainnet                                      |
| **总供应** | 1,000,000,000                                     |
| **用途**   | 治理投票、质押(Stake-to-Earn)、POI 奖励、生态门禁 |
| **定位**   | 网络级价值单元，不是公司股权或 RWA 收益凭证       |

### SSFShareToken

| 属性       | 值                                   |
| ---------- | ------------------------------------ |
| **类型**   | ERC-20                               |
| **网络**   | Base Mainnet                         |
| **总供应** | 20,000                               |
| **发行价** | 1,000 USDC                           |
| **用途**   | 代表 SF South Bay 项目的经济权益份额 |
| **定位**   | 项目级代币，需合格投资者审核         |

## 2. 功能凭证 (Utility Credentials)

### Access Pass（访问通行证）

- **用途**: 项目活动/功能的访问凭证
- **发行**: 系统根据规则自动发行
- **可转让**: 否

### Showing Pass（看房通行证）

- **用途**: 现场看房预约凭证
- **发行**: 通过邀请或审核
- **可转让**: 否

### Service Credits（服务积分）

- **用途**: 平台内服务抵扣
- **定位**: $POI 直接作为 Service Credits 使用
- **说明**: 无需独立代币，POI 即是统一消费单位

---

## 3. 合规凭证 (Compliance Credentials)

### KYC Credential

- **用途**: 身份验证完成凭证
- **发行**: 通过 KYC 审核后自动发行
- **链上**: 可选（SBT 或链上标记）
- **隐私**: 仅标记"已通过"状态，不存储 PII

### Vendor Credential

- **用途**: 供应商/合作方资格凭证
- **发行**: 通过商户入驻审核
- **可转让**: 否

---

## 4. 证据链凭证 (Evidence Chain Credentials)

### Receipt Token（回执凭证）

- **用途**: 交易/操作的不可篡改回执
- **内容**: 订单号、金额、条款版本、披露确认
- **存储**: 链上或离线（SHA256 校验）
- **功能**: 审计追溯、争议解决

---

## 5. 生态门禁 (Ecosystem Access)

### Priority Pass

- **用途**: 通过 POI 质押获得的优先级提升
- **权益**: 选房优先、折扣、Cap Boost
- **获取**: POI 质押 → 自动计算优先级等级
- **说明**: 非独立代币，而是 POI 质押状态的衍生权益

---

## 6. 代币定位矩阵

| 代币/凭证       | 可交易          | 可转让 | 代表权益    | 需 KYC         |
| --------------- | --------------- | ------ | ----------- | -------------- |
| $POI            | ✅ DEX          | ✅     | 治理 + 激励 | 部分功能需要   |
| SSFShareToken   | ❌ (无二级市场) | 受限   | 经济权益    | ✅             |
| Service Credits | N/A             | N/A    | 服务消费    | N/A            |
| KYC Credential  | ❌              | ❌     | 身份凭证    | 本身即 KYC     |
| Receipt Token   | ❌              | ❌     | 交易回执    | 取决于交易类型 |
| Priority Pass   | ❌              | ❌     | 优先级      | 否             |

---

> 所有代币与凭证的权利与限制以签署文件为准。功能性定位，不构成证券或收益承诺。
