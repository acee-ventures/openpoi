# $POI 代币经济学

> Proof of Influence — 网络级价值代币经济学框架

---

## 1. 代币概览

| 参数       | 值                                                      |
| ---------- | ------------------------------------------------------- |
| **名称**   | $POI (Proof of Influence)                               |
| **总供应** | 1,000,000,000 $POI (10 亿)                              |
| **网络**   | Base (Testnet: Sepolia / Mainnet: 8453)                 |
| **标准**   | ERC-20 (ERC20Votes, UUPS Upgradeable)                   |
| **定位**   | 网络级价值单元：治理投票、质押(Stake-to-Earn)、生态门禁 |

> $POI 不是公司股权，不代表 RWA 收益分成，不作为直接支付手段。

---

## 2. 初始分配方案

| 分配类别               | 比例 | 数量        | 用途                                 | 释放节奏                 |
| ---------------------- | ---- | ----------- | ------------------------------------ | ------------------------ |
| **社区激励池**         | 40%  | 400,000,000 | 任务/贡献/Stake-to-Earn              | 按激励计划逐步释放       |
| **RWA 项目激励池**     | 25%  | 250,000,000 | SSF 等 RWA 项目激励（系列级上限 5%） | 按项目激励计划释放       |
| **DeFi/Onramp 激励池** | 15%  | 150,000,000 | DeFi 功能激励、Onramp 奖励           | 按功能上线节奏释放       |
| **团队与顾问**         | 10%  | 100,000,000 | 团队与顾问激励（Vesting）            | 4 年线性释放，1 年 Cliff |
| **储备池**             | 10%  | 100,000,000 | 储备（治理、应急、生态发展）         | 治理决策释放             |

---

## 3. 用途

| 用途              | 说明                                                |
| ----------------- | --------------------------------------------------- |
| **治理投票**      | ERC20Votes 机制，参与协议治理                       |
| **Stake-to-Earn** | 质押获取各类项目激励（按项目系列级上限分配）        |
| **生态门禁**      | 质押状态产生优先级提升（选房优先、折扣、Cap Boost） |
| **服务消费**      | 作为 Service Credits 消费平台服务                   |

> POI 仅通过质押 (stake) 产生权限，不直接用于购买 RWA 产品。

---

## 4. RWA 项目系列级上限

| 项目                   | 上限占比 | 上限数量        |
| ---------------------- | -------- | --------------- |
| **SSF (SF South Bay)** | 5%       | 50,000,000 $POI |

- 每个 RWA 项目系列均有独立上限
- 系统在累计分配达到上限时强制 hard stop
- 各项目具体激励公式详见对应 tokenomics 文档
